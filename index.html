How it works
3단계면 끝입니다
레포지토리에서 프로덕션까지 1분 이내.
1

GitHub URL 입력

GitHub 레포지토리 링크를 붙여넣으세요. Public과 Private 모두 지원합니다.

2

Deploy 클릭

Nixpacks가 Node, Python, Go, Ruby 등 스택을 자동 감지하고 빌드합니다.

3

라이브 URL 수령

앱이 Public URL과 HTTPS로 배포됩니다. DNS 설정 없이 바로 공유하세요.

Features
배포에 필요한 모든 것
설정이 아닌 배포에 집중할 수 있도록 만들었습니다.

한국 리전 (AWS 서울)

AWS ap-northeast-2에서 운영. 국내 사용자에게 최적의 응답 속도를 제공합니다.

20+ 언어 자동 감지

Node.js, Python, Go, Ruby, Java, Rust 등 Nixpacks가 자동으로 감지하고 빌드합니다.

자동 HTTPS

모든 앱에 Let's Encrypt SSL이 자동 적용됩니다. 별도 설정 불필요.

무료로 시작

앱 2개까지 영구 무료. 신용카드 없이 바로 시작하세요.

모니터링 & 알림

CPU, 메모리, 디스크 사용량 실시간 추적. 앱 상태 변경 시 알림.

Git Push 자동 배포

GitHub Webhook 연동으로 git push만 하면 자동으로 재배포됩니다.
